# Builderall AppShell

## Development

```shell
npm install
npm start
```

## Production

```shell
npm ci
npm run build
```

## How to use

```shell
npm install builderall.appshell
```

```html
<body>
  <builderall-appshell></builderall-appshell>
  <script type="module" src="./scripts/index.ts"></script>
  <div class="app_content">
      <!--  YOUR CONTENT HERE  -->
  </div>
</body>
```
